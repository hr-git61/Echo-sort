Echo sort (File System Access API 版) の使い方

  • 初回は画面右下に「保存先を選ぶ」バナーが出ます。クリックして保存先フォルダを指定してください。
  • 以降、manaba の「ファイルっぽい」リンクをクリックすると、manaba/科目名/元ファイル名 に自動保存されます。
  • 保存先を変えたいときは、manaba ページ上で Alt + クリック。

ブラウザ設定で「サイトにファイルとディレクトリへのアクセスを許可」が必要です。
取得には fetch(..., { credentials: 'include' }) を使用します。事前に manaba にログインしてください。
